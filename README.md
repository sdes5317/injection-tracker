# Mounjaro 腹部注射追蹤器

追蹤 Mounjaro (tirzepatide) 腹部皮下注射位置的輕量網頁工具，幫助你輪替注射部位、避免重複施打同一區域。

## 功能總覽

### 注射位置追蹤
- 腹部分為 **4 個象限**（左上、右上、左下、右下），點擊即可記錄注射
- 每個象限以顏色標示推薦程度：
  - 🟢 **綠色（推薦）**：距上次注射已超過 28 天，可安全施打
  - 🟡 **黃色（尚可）**：距上次注射 14–28 天
  - 🟠 **橙色（注意）**：距上次注射 7–14 天
  - 🔴 **紅色（避免）**：距上次注射不足 7 天
- 肚臍周圍標示為 **禁區**，不可施打

### 象限資訊顯示
- 每個象限直接顯示距上次施打的天數（如「3天前」、「推薦」）
- 同一象限施打超過一次時顯示累計次數

### 左右手標示 & 鏡像翻轉
- 小熊兩側標示左手 ✋ / 右手 🤚 方向
- 點擊「🔄 鏡像翻轉」可切換左右手方向（適應不同慣用手）
- 鏡像模式下，四個象限的顏色、天數、點擊記錄會同步翻轉

### 注射記錄
- 點擊象限後彈出表單，可填寫：
  - **日期與時間**（預設為當前時間）
  - **劑量**（2.5 / 5 / 7.5 / 10 / 12.5 / 15 mg）
  - **備註**（選填）
- 若該象限在 7 天內已施打過，會顯示警告提醒
- 劑量自動沿用上一次的設定

### 統計面板
- **距上次注射**：顯示天數
- **下次建議**：依 7 天週期計算下次施打時間
- **總注射次數**：累計記錄筆數

### 注射歷史
- 以時間倒序列出所有記錄
- 顯示日期、象限位置、劑量、天數、備註
- 最多顯示 4 筆，超過可滾動瀏覽
- 支援逐筆刪除（含確認對話框）

### 資料管理
- **自動儲存**：所有資料存於瀏覽器 `localStorage`
- **匯出 JSON**：將記錄匯出為 JSON 檔案備份
- **匯入 JSON**：從 JSON 檔案匯入記錄（自動去重）
- **清除全部**：一鍵清除所有記錄（需確認）

## 技術架構

- 純前端靜態網站（HTML + CSS + JavaScript），無需後端
- 適用 GitHub Pages 部署
- 響應式設計，支援桌面與手機瀏覽
- SVG 即時渲染象限覆蓋層
- 支援觸控操作

## 部署方式

1. 將專案推送到 GitHub 倉庫
2. 進入 Settings → Pages
3. 選擇 `main` 分支，點擊 Save
4. 等待部署完成後即可透過 `https://<username>.github.io/<repo>/` 存取
